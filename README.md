# DINKIssTyle Chrome Extensions

A collection of Chrome extensions that I created because I might need them, and perhaps someone else does too.

This repository contains various unpacked Chrome extensions designed for specific utility purposes.

## How to Install

Since these are unpacked extensions, you need to install them via Chrome's Developer Mode.

1.  **Download or Clone** this repository to your local machine.
2.  Open **Google Chrome**.
3.  Navigate to `chrome://extensions` in the address bar.
4.  Toggle **Developer mode** on in the top right corner.
5.  Click the **Load unpacked** button in the top left.
6.  Select the specific folder of the extension you want to install (e.g., `Image-Information` or `save-image-as-type`).
    *   *Note: Do not select the root folder of this repository. Select the individual folder for the extension you wish to use.*
7.  The extension is now installed and ready to use!

## Extensions List

*   **Article-View**: Extract articles and posts into a clean, ad-free reader window with zoom controls, font toggle (Serif/Sans), theme modes (Light/Dark/Auto), advanced settings (custom font, line spacing), fullscreen, and Save as PDF.
*   **Image-Information**: Right-click on any image (even those behind layers) to view detailed metadata like filename, MIME type, size, and dimensions.
*   **LM-Studio**: Process selected text with a local LLM via LM Studio API. Features chat-style bubble UI, conversation history, streaming responses, and customizable prompts.
*   **save-image-as-type**: Context menu options to save images as different formats (JPG, PNG, WebP) or the original.
*   **Visual-Effects**: Apply various visual effects and page transitions to web pages via a menu.
*   **Zoom-To**: Double-click on empty space within a content container to smart zoom the entire page to fit that content using native browser zoom. Double-click again to restore.

# DINKIssTyle 크롬 확장 프로그램

제가 필요해서, 그리고 다른 사람들도 필요할지도 몰라서 만든 크롬 확장 프로그램 모음입니다.

이 저장소에는 특정한 유틸리티 목적을 위해 설계된 다양한 압축 해제된(unpacked) 크롬 확장 프로그램들이 포함되어 있습니다.

## 설치 방법

이것들은 압축 해제된 확장 프로그램이므로, 크롬의 개발자 모드를 통해 설치해야 합니다.

1.  이 저장소를 로컬 컴퓨터에 **다운로드하거나 복제(Clone)**합니다.
2.  **구글 크롬**을 엽니다.
3.  주소창에 `chrome://extensions`를 입력하여 이동합니다.
4.  우측 상단 모서리에 있는 **개발자 모드(Developer mode)**를 켭니다.
5.  좌측 상단에 있는 **압축해제된 확장 프로그램을 로드합니다(Load unpacked)** 버튼을 클릭합니다.
6.  설치하려는 확장 프로그램의 특정 폴더(예: `Image-Information` 또는 `save-image-as-type`)를 선택합니다.
    *   *참고: 이 저장소의 최상위 루트 폴더를 선택하지 마세요. 사용하려는 확장 프로그램의 개별 폴더를 선택해야 합니다.*
7.  이제 확장 프로그램이 설치되었으며 사용할 준비가 되었습니다!

## 확장 프로그램 목록

*   **Article-View**: 기사와 게시물을 광고 없는 깔끔한 리더 창으로 추출합니다. 줌 조절, 글꼴 전환(세리프/산세리프), 테마 모드(라이트/다크/자동), 고급 설정(사용자 지정 글꼴, 줄 간격), 전체 화면 및 PDF 저장 기능을 제공합니다.
*   **Image-Information**: (레이어 뒤에 있는 이미지 포함) 이미지 위에서 우클릭하여 파일명, MIME 타입, 크기, 치수와 같은 상세 메타데이터를 확인합니다.
*   **LM-Studio**: 선택한 텍스트를 LM Studio API를 통해 로컬 LLM으로 처리합니다. 채팅 스타일 버블 UI, 대화 히스토리, 스트리밍 응답, 사용자 정의 프롬프트 기능을 제공합니다.
*   **save-image-as-type**: 우클릭 메뉴 옵션을 통해 이미지를 다양한 형식(JPG, PNG, WebP) 또는 원본으로 저장합니다.
*   **Visual-Effects**: 메뉴를 통해 웹 페이지에 다양한 시각 효과와 페이지 전환 효과를 적용합니다.
*   **Zoom-To**: 콘텐츠 컨테이너 내의 빈 공간을 더블 클릭하면 브라우저의 기본 줌 기능을 사용하여 해당 콘텐츠에 맞춰 페이지 전체를 스마트 줌 합니다. 다시 더블 클릭하면 복원됩니다.
